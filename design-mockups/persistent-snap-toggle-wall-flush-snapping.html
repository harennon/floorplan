Persistent snap toggle + wall-flush snapping

Issue #29 · Two things: a discoverable, persisted grid-snap on/off control, and symbols that seat flush against a nearby wall face. Alt still bypasses snapping transiently. Wall-flush beats the nearest grid line.

Part 1 · Snap toggle — three placement options
Option A · HUD toggle cell recommended
Zoom 100%

A new interactive HUD cell sits directly above the existing precision chip. One tap toggles; the filled pip + gold text mean On. Lives in the same stack as the Grid/Scale/Zoom cells — zero new screen real estate.

Option B · Segmented On/Off pill variant
Snap
Zoom 100%

An explicit two-state pill echoing the top-right unit toggle. More self-evident (both states visible) but wider and visually heavier than the single cell.

Option C · Magnet button on tool rail variant

A magnet toggle on the existing left rail (a common CAD idiom). Strong metaphor, but the rail is drawing-tool focused and hidden/collapsed on mobile, so the control would be less reachable while placing furniture.

Recommended toggle — the three states

Default. Filled gold pip.

Persisted off. Hollow pip, muted.

Alt held — transient bypass while On.

The toggle owns the persisted preference (survives reload via localStorage). Holding Alt is a momentary override that does not change the stored value — the cell shows a dashed "Alt" hint while held, then reverts.

Part 2 · Snapping behaviour when placing / dragging a symbol
1 · Grid snap (Snap On, no wall near)

Away from walls the symbol lands on the nearest grid line — today's behaviour, unchanged. A teal grid-snap dot marks the target.

2 · Wall-flush snap the new value

When the symbol's nearest edge is within threshold of a wall face, that edge snaps flush against the wall. A green guide line highlights the wall face; the seated edge glows green.

3 · Precedence: wall beats grid

When a grid line and a wall face are both in range, the wall wins. The faint teal grid dot shows where grid alone would have landed; the symbol instead seats on the green wall face.

4 · Snap Off / Alt held → raw point

With the toggle Off (or Alt held), the symbol drops at the exact pointer position — no grid, no wall-flush. A muted ✕ marks the raw drop point.

Interaction model & feedback

Cursor-side snap tag reuses the existing .snap-tag element to name the active snap so the behaviour is legible: flush grid free. Green = wall-flush, teal = grid, muted = free. Guide lines fade in over ~120ms and clear the instant the edge leaves threshold, honouring prefers-reduced-motion.
