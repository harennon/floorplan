Living / dining pieces

Armchair · Coffee table · Round dining table — added to the Living dock tab, reusing the existing symbol pattern exactly. This mockup shows the dock icons, the to-scale on-canvas glyphs, selected state with dim chips, and rotation behaviour of the circular table over its rectangular footprint.

1 · Living dock tab — with the 3 new items

The three additions sit alongside the existing living pieces in #row-living. No new category tab. New items are flagged with a teal dot here for the review only.

2 · On-canvas glyphs, drawn to scale (60 px / m)

Each glyph is composed only from existing SVG primitives (rect, line, circle, ellipse, polygon) painted inside the w×h footprint — the same machinery as every other symbol. Existing pieces shown greyed for scale reference.

The round dining table's footprint stays a w×h box for selection, dim chips, hit-test and clearance — only the drawn glyph is circular. Selection box (dashed), rotate handle, and both dim chips render exactly like every other symbol.
