Fireplace — a wall-anchored focal element

Adds a fireplace type to the living catalog so the interior-design skill can place and orient seating toward a real fireplace — the top focal priority (fireplace > TV > window > blank wall). Wall-anchored, shallow depth, per-axis bounds + named presets like every other catalog entry.

This mockup shows: the new dock button glyph (3 options), the on-canvas symbol render (default + rotated/flush-to-wall, with a sofa oriented toward it), and the inspector preset chips.

1 · The dock — new button in the Living row

The fireplace button drops into #row-living next to tv / bookshelf. It reuses the generic .dock-item markup — data-type="fireplace" is the only load-bearing attribute; placement/select/resize are automatic. A new badge marks it on first ship (CSS-only, matches the rug precedent).

2 · Glyph direction — pick one

All three read at 18px in the dock and reuse currentColor (gold on hover). The only decision is whether the flame gets a warm --ember accent (the sole non-monochrome glyph in the catalog) or stays monochrome like every existing dock icon.

A · Mantel + ember flame recommended

Mantel top rail + recessed hearth opening + a single ember flame. Instantly reads "fireplace" and the warm accent flags it as the focal element. Left tile = resting (ink), right = hover (gold).

B · Monochrome hearth

Same silhouette, flame drawn as an outline in currentColor. Perfectly consistent with the existing monochrome dock set — zero new palette token. Slightly less "at-a-glance focal" than A.

C · Flame-only mark

Just the flame, no hearth. Boldest read but risks colliding with "candle/heat" meaning and drops the wall-anchored cue. Not recommended for a plan-view furniture piece.

3 · On the canvas — plan-view render + focal orientation

Plan-view interior glyph (proposed for _renderInterior): a shallow hearth box flush to the wall with a small ember mark, so it reads as a fireplace from above and is distinct from a tv or bookshelf. The sofa's front faces the fireplace — exactly the focal orientation the skill computes. Selected fireplace shown with the gold dashed selection box + rotate handle + dim chips.

4 · Proposed catalog data (for the LLD)

Wall-anchored, shallow depth, continuous resize (like sofa/tv — not discrete, not openings, not circular). Category living. Dimensions grounded in real hearth/insert widths (30" insert → 60" wide) and shallow projection (~0.25–0.45 m). Presets sit within the per-axis bounds.
