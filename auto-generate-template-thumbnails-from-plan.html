Auto-generated template thumbnails

Each gallery card's preview is rendered from its plan data by a shared static SVG renderer (walls + room fills + symbol footprints), fit into the thumbnail box. No hand-authored thumb SVG. Adding a template needs only a valid plan.

What the renderer draws

  • Room fills + walls — closed-room polygons filled at low alpha, wall body + crisp centerline, mirroring exportImg.js.
  • Symbol footprints — every symbol's rotated bounding box. Detailed glyphs additionally reuses the symbolRender.js interior marks (bed pillow, sofa back, door swing, stove burners…). Flat footprints draws boxes only — cheapest, always correct.
  • Auto-fit — world bounds + margin drive the SVG viewBox; the box is preserveAspectRatio centered, so any plan proportion fits the 4:3 card without distortion.
  • No chrome — no dimension labels, grid, snap glyphs, or type text. It is a silent preview, not an export.
