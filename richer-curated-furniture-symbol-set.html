Richer curated furniture & symbol set

floorplan · palette layout & glyph exploration · issue #43

Today's dock is a flat horizontal strip with 8 items in 2 groups. Growing to ~20 curated household symbols across 5 categories needs a palette that stays uncluttered on both desktop and a phone. Below: three dock layouts to choose between, then the proposed glyph set with default sizes and clamp bounds.

Proposed symbol set — glyphs, defaults & clamp bounds

Existing 8 types are marked ●. New types render as recognizable, to-scale glyphs in symbolRender.js and gain a label, category, default w×h, and min/max following the clampDim model. Glyphs are shown to scale relative to each other (metres → px).

Option A — Category tabs (recommended)

A tab strip along the top of the dock; one category's items shown at a time in a single row. Scales to any number of categories, always low visual density, and the active category label is always visible. Tap a tab, then drag an item onto the canvas. On mobile the tab strip and the item row each scroll horizontally if needed.

select mode · drag an item onto the canvas
Option B — One long strip, grouped

The current pattern extended: every symbol in one horizontally-scrolling row with vertical group dividers and rotated category labels. Zero new interaction to learn, but ~20 items means a lot of scrolling on a phone and no way to jump to a category.

select mode · scroll the strip, drag an item
Option C — Compact bar + “More” grid drawer

A short bar shows the most-used items (openings + a few staples) plus a More button that opens a categorized grid drawer above the dock. Keeps the resting footprint tiny — great on mobile — at the cost of one tap to reach less-common items. Click More below to open the drawer.
